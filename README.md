# Freeze Food ERP

Freeze Food ERP is ERP for sea food management. It has Module such as Request Document, Purchase Order Document, Receive Supplier Document, Delivery Note Document, Receive Note Document.

## Installation


```bash
npm install
```

## Usage

```python
npm run dev
```
